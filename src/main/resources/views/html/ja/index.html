なぜMultiBit HDを選ぶのか？

マルチビットHDは次のとおりです。

  • 迅速かつすべての主要なデスクトップオペレーティングシステム（Windows、OS X、Linux）の全体にインストールするのは簡単
  • 設計によってセキュア - すべてのデータがローカルマシン上で暗号化されたまま
  • より高速に同期化します - 通常分程度を
  • 決定論 - あなたはあなたのbitcoinを再利用するために使用できるシードフレーズを持っている
  • 分散型 - なし固定サーバーのみBitcoinネットワーク
  • （前回のリリースで40以上）複数の言語 - 国際

あなたがBitcoinの通常のユーザーであり、単純に手間をかけずに動作する財布をする場合は、マルチビットHDはあなたのための正しい選択です。
