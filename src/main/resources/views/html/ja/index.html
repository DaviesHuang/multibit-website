なぜマルチビットHDを選ぶのか？

MultiBit HD is:

  •  迅速かつ簡単にインストールする
    私たちは、すべての主要なデスクトップオペレーティングシステム（Windows、OS X、Linux）をサポートして
  •  設計によるセキュア
    すべてのデータは、ローカルマシン上で暗号化保たれ、我々はそのようなTrezorなどのハードウェア財布をサポートされている
  •  決定論的な
    シード·フレーズは、データの損失からあなたのビットコインを保護
  •  分散型
    いいえ、固定サーバーのみビットコインネットワーク
  •  インターナショナル
    私たちは、40以上の言語をサポートしてい

あなたはビットコインの通常のユーザーであり、単純に手間をかけずに動作します財布をする場合は、マルチビットHDはあなたのための正しい選択です。

あなたは上記のボタンを使用して、それをダウンロードし、起動して5分でビットコインを使用して実行することができます。
