Getting started

To install MultiBit on your computer, follow the steps on the main help page.

When you first open a new installation of MultiBit a new, empty wallet is created for you in your application data directory with the name 'multibit.wallet'. There are more details on how you can manage your wallets in the Wallet management help page.

When you first open your installation of MultiBit it will begin downloading data from the bitcoin network. It downloads the 'block chain' which contains details of all the verified transactions in the system.
