Getting started

MultiBit HD is designed to be easy to use. It is intended for domestic (or small scale merchant) users who wish to use a desktop computer for their day to day Bitcoin transactions.

This guide will provide you with the first steps you should take after you have installed MultiBit HD so that you can be confident about how to get things done safely.

Step 1 - Download and install MultiBit HD

We use Install4j to create our installers so that we can support the widest range of environments.

After downloading the installation program, simply double click the icon and follow the on-screen instructions. In most cases just accepting the defaults will be sufficient.

The "How to Install" guide provides more detailed instructions if you need them.
