Edit Wallet

To change your wallet's name, or where you store its backups, select "Tools" and then select "Edit wallet".

  1. Type over the wallet name and notes.
  2. Click on the file selector icon to choose the cloud backup folder.
  3. Click "Apply" to set the new name, notes and cloud backup folder.
  4. You can always change the wallet features by coming back to the "Edit wallet" screen.

Some additional information

The wallet name and notes are shown at startup.

The cloud backup location sets the local folder on your computer where backup copies of your wallet are made.

If you restore a wallet from seed, then the default notes will show the wallet as having been "created" on the date you restored it.
