File descriptions

When MultiBit HD runs, it creates various files on your computer. This help section describes what each of these files is for.

Here is a list of the files stored with your wallet:

  • mbhd.wallet.aes  =  This is the main wallet file containing your private keys and transactions.

  • mbhd.yaml  =  This file contains wallet configuration information.

  • mbhd.checkpoints  =  A Bitcoin checkpoints file, used in syncing your wallet.

  • mbhd.spvchain  =  A Bitcoin blockstore, used in syncing your wallet

Contacts. In the ‘contacts’ directory is stored a file ‘contacts.aes’. This contains your contacts information.

Payments. In the ‘payments’ directory is stored a file ‘payments.aes’. This contains your payments information.

History. In the ‘history’ directory is stored a file ‘history.aes’. This contains your history information.

Rolling backups. In your rolling backup directory you will see files with the name ‘mbhd-YYYYMMDDHHMMSS.wallet.aes’. These are rolling wallet backup files. Every time the wallet writes to disk, the existing wallet is kept as a backup. The main purpose of this file is to recover from any sudden loss of power that prevents a clean wallet save. The last four rolling backups are kept.

Local zip backups. In the directory ‘zip-backup’ are stored encrypted zip copies of all the files in your wallet directory.

Cloud backups. If you have a cloud backup directory set for your wallet, the local zip backups are regularly copied to your cloud backup directory.
