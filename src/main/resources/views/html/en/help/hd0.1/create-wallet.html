Create Wallet

When you start MultiBit HD for the first time you will be shown the Welcome Wizard which will guide you through the process of creating a new wallet. If you want to create an additional wallet you can use the "Restore" button from the "Enter password" screen which will bring up the Welcome Wizard starting at the "Select Create Wallet Option" screen.

This help article will guide you through the Welcome Wizard from the beginning.

Accept the licence

The MultiBit HD licence is very permissive and is based on the MIT licence. If you do not accept the licnce then you must Exit the application.

Select a language

MultiBit HD benefits from a large number of translators who have generously given of their time and expertise to make your experience of using the application as natural as possible. If you would like to contribute to this translation effort please visit our translation site.

After selecting your language the screen will immediately refresh to use it.

Select "Create Wallet" option

At this point you can choose to switch to a pre-existing wallet or restore one from a seed phrase. In this article we are creating a wallet for the first time so the appropriate option is "Create Wallet" as shown.

Make preparations

Creating a Bitcoin wallet is a serious business and we strongly recommend that you give yourself some time to familiarise yourself with the steps involved before proceeding.

In short, the process is:

  1. Select a backup location
  2. Write down a seed phrase and a timestamp
  3. Confirm that you have written down the information correctly
  4. Provide a password for day to day use
