Exchange settings

Which fiat currency do you want to use to illustrate the fiat value of your transactions and whose rate do you want to use to estimate the fiat value? To choose the source of the exchange rate, select "Preferences" and then select "Exchange".

  1. Select the exchange you want to use.
  2. Select the currency you want to use from those offered by the exchange.
  3. Click "Apply" to use this exchange's rates for the fiat value of future transactions.

Some additional information

You can fine tune how the fiat currency is displayed in the units screen. Whatever currency and units you choose will determine how the transactions are shown in the payments screen.
