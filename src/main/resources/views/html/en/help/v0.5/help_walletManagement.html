Wallet management

With MultiBit, you can have as many wallets as you like. They are usually given the file extension ".wallet".

When you open MultiBit, the wallets that were open in the previous use of MultiBit are reopened. If this is the first time you have opened MultiBit a new, empty wallet file called "multibit.wallet" is created in your user data directory.

Your wallets appear in the side panel labelled " Wallets". Click on one of the wallets to make it the active wallet. The balance for the active wallet is shown in the top left of the MultiBit screen. The wallet description and file location are shown in the title of the MultiBit window.

Create a new wallet

To create a new wallet, click on the " New Wallet" button or choose the menu option "File |  New Wallet". This opens a dialog window where you choose the location and file name of the new wallet. Click "Ok" and a new wallet is created with the default description "Your wallet description".

Rename the wallet

To rename the wallet, simply click on the descriptive text and type a new description. This change will be saved in the "multibit.info" file in the application directory when either the wallet is closed, or MultiBit is shut down. This does not change the file name of the wallet.

Open a wallet

To open another wallet, choose the "File |  Open Wallet" menu option. This opens a file chooser dialog where you can choose the wallet to open. When the wallet is opened MultiBit automatically synchronises it to get the latest transactions.

Note that MultiBit cannot open the wallet files from the Satoshi bitcoin client (wallet.dat files).

Close a wallet

To close a wallet, choose the "File |  Close Wallet" option. This saves and closes the wallet and it is no longer shown in MultiBit. It is still stored on your disk - it is not deleted.

Delete a wallet (advanced)

Deleting a wallet is an irreversible action known as a file "shred". If the addresses associated with the private keys in the wallet have ever been made public there is the chance that someone could use them to try to send you bitcoin in the future. Without the private keys you will have no way to spend the bitcoin.

That said, if you do not expect to retain full control over your computer (such as selling it) you may need to be sure that the wallet is unrecoverable by a malicious third party. If so, then make sure that you retain encrypted backups of your private keys.

There is a hidden menu option to enable the menu item that allows you to delete a wallet. It is not shown by default to avoid accidental loss of wallets and is intended for more experienced users. For details of how to enable this option, read the 'configuration.md' file in the MultiBit installation directory.
