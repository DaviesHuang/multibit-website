What are the different types of wallets ?

There are two types of wallets in MultiBit. They are:

  • Non password protected wallets. They are shown with the icon ''.
  • Password protected wallets. Their icon is ''.

Password protected wallets are much more secure than non password protected ones. It is recommended that you password protect all MultiBit wallets that contain more than a little 'loose change'. You can add a password to a wallet using the 'File | Add Password' menu option.

You need to use a strong password.

Do not lose the password that you use for a wallet. You need your password to spend your bitcoin. There is no way to recover your password if you lose it.
