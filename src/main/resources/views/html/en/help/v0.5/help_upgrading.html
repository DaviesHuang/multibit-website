Upgrading MultiBit

Every so often a new version of MultiBit will be released containing bug fixes and new features.

This guide will provide you with the correct process to follow when you decide to upgrade to a newer version. It is assumed that you have reviewed the Getting Started guide and know where all your wallet files are.

Step 1 - Locate your wallet files

If you have a standard installation upgrading MultiBit should be a seamless process. Your wallet and configuration files will not be in the same directory as the MultiBit application and will be in a location that is regularly backed up.

In a non-standard installation your wallet and configuration files will be in the same directory as the MultiBit application. This would likely cause the new version of MultiBit to ignore the previous wallets and configuration because it is not aware of them.

To avoid this situation, MultiBit recommends that you make sure that your wallets and configuration files are always kept separate from the installation directory.

You can always see the file location of your current wallet in the main title bar of your MultiBit window. For instance, it might say: "MultiBit - My wallet - /Users/jim/Library/Application Support/MultiBit/multibit.wallet". The file name at the end is where your wallet is stored.

Applications are usually stored in C:\Program Files on Windows or /Applications on Mac.

Step 2 (optional) - Move your wallets

If you have a non-standard installation and want to move the wallets out of the MultiBit installation directory then please review the Moving A Wallet article.

Step 3 - Run the installer

Download the installer from the official MultiBit site and verify the GPG signature to be absolutely sure you have a legitimate copy.

Follow the standard installation instructions for your computer.

Finish

That's all there is to it. MultiBit will alert you if there is a problem and the main help contents provide answers to many common questions.
