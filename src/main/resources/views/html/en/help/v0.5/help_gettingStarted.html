Getting started

When you first open MultiBit a new, empty wallet is created for you called 'multibit.wallet'.

MultiBit also starts downloading the data it needs from the Bitcoin network. This normally takes a minute or so.


If you are a new user of Bitcoin, there are a couple of safety features that you should be aware of:

  • If you have substantial funds in a wallet: Password protect your wallet.

  • In case you lose your computer: Backup your wallet.
