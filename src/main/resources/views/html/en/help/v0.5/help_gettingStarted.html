Getting started

When you first open a new installation of MultiBit a new, empty wallet is created for you in your user data directory called 'multibit.wallet'.

When you first open MultiBit it will begin downloading data from the Bitcoin network. It downloads a part of the 'block chain' that contains all the bitcoin transactions.


If you are a new user of Bitcoin, there are a couple of safety features that you should be aware of:

  • If you have substantial funds in a wallet: Password protect your wallet.

  • In case you lose your computer: Backup your wallet.
