Add / Change / Remove password


Do not forget your wallet password.

There is no 'password reset'.


Add Password

To add password protection to a wallet:

  1. Choose the menu option 'File |  Add Password'.

  2. Enter the password you want to use in both the 'Password' and 'Repeat password' fields. If the two passwords match you will see a green tick (  ) to the right of the password fields.

  3. Press the ' Add password to wallet' button. You will see a success message when this completes successfully.


Change Password

To change the password on a wallet:

  1. Choose the menu option 'File |  Change Password'.

  2. Enter the current wallet password in the field 'Current Password' | 'Password'.

  3. Enter the new password you want to use in both the 'New password' and 'Repeat new password' fields. If the two passwords match you will see a green tick (  ) to the right of the password fields.

  4. Press the ' Change password' button. You will see a success message when this completes successfully.


Remove Password

To remove the password protection from a wallet:

  1. Choose the menu option 'File |  Remove Password'.

  2. Enter the current wallet password in the field 'Password'.

  3. Press the ' Remove password' button. You will see a success message when this completes successfully.
