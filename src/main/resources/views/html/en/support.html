Fast fixes

Sometimes a transaction seems to get "stuck" leading to concern that it hasn't been sent or received. This is almost always fixed by using the "Reset Blockchain" option from the Tools menu.

Check the release documentation for your version.

Reporting a bug

If you discover a bug in MultiBit we welcome feedback - it makes MultiBit better for everyone. We are introducing an automatic error reporting system that can be used to send us information about what went wrong (subject to your consent and keeping your privacy very much in mind). We expect this to be available in v0.5.x

Until then, the manual bug reporting procedure is as follows:

  1. Copy the following text: 
      Bug report
  Operating System:
  MultiBit Version:
  Steps to replicate the problem:
  2. Visit the MultiBit Issue reporting center and perform a quick search in the "Issues & Milestones" box to check that no-one else has reported the same issue as you.
  3. Assuming yours is unique (no point flooding us with duplicates) then click on "New Issue".
  4. Paste the template text and fill it in for your system. The more detail you can give the quicker we can address the problem.

Due to the number of downloads of MultiBit now gets we cannot answer every support query personally. We do see these issues and make acknowledging them a priority.
