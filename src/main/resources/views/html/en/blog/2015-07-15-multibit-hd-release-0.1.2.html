MultiBit HD Release 0.1.2 is now available

We are pleased to announce the next MultiBit HD Release (0.1.2) is available for download.

This is a maintenance release.

Summary of fixes

Here is a quick summary of the fixes contained in this release:

  • Miners' fee slider range increased in light of recent network spam
  • History functionality has been removed as it is largely replicated by the 'Report error' functionality
  • Various small user interface tidy ups

Semi-automatic update

We normally wait 24 hours after a release and then activate the semi-automatic upgrade process. You decide if the upgrade occurs. The upgrade process will not affect your wallet data. Only the application binaries will be upgraded.

You can find more details, including digital signatures, on the download page.
