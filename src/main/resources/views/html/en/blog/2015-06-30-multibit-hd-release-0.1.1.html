MultiBit HD Release 0.1.1 is now available

We are pleased to announce that our first maintenance release MultiBit HD Release 0.1.1 is available for download.

This release fixes a few problems that people have kindly reported through our Error Reporting button.

Summary of fixes

In no particular order, here is a quick summary of the fixes contained in this release:

  • Faster startup
  • Improved colour scheme
  • Mining dues not being handled correctly

Semi-automatic update

On startup MultiBit HD will contact the MultiBit.org server to determine if it has the latest version installed and then present an upgrade dialog if a later version exists. You can choose to cancel the update if it is inconvenient, or you can go ahead by following the on-screen instructions.

The upgrade process will not affect your wallet data. Only the application binaries will be upgraded.

You can find more details, including digital signatures, on the download page.
